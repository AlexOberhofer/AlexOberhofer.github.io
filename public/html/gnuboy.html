SDL2 GNUBoy Port



GNUBoy

GNUBoy is historically a Gameboy and GameBoy Color emulator developed in the early 2000s. The system could run on many systems and was very fast. I forked an updated version of the old codebase and ported a large amount of the existing SDL code to SDL2. I have also now been able to expand the codebase to build on Windows and Mac, as it originally only built for unix systems.
